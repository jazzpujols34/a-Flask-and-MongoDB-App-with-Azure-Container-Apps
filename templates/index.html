MongoDB + Flask + Azure App Services Demo

This tutorial will show you how to create a functional Flask application that connects to a MongoDB Atlas database, and implements CRUD (Create, Read, Update, and Delete) functionality:

  • Create a database in MongoDB Atlas
  • Set up and understand how to use Flask
  • Use your database as your apps backend
  • Create a CRUD app using Flask, Python, and Postman
  • Host your complete Flask app on Azure
